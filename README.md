ApiYummy, modern restoran yönetim ihtiyaçlarını karşılamak üzere geliştirilmiş, ASP.NET Core Web API tabanlı ve Yapay Zeka (LLM) entegrasyonuna sahip kapsamlı bir yönetim sistemidir. Proje, rezervasyon yönetiminden anlık veri analizine, AI destekli tarif üretiminden modern dashboard tasarımlarına kadar geniş bir yelpaze sunar.

🚀 Öne Çıkan Özellikler
AI Şef Modülü: GroqCloud API (LLM) kullanılarak, eldeki malzemelere göre saniyeler içinde özgün yemek tarifleri üreten yapay zeka entegrasyonu.

Dinamik Dashboard: Chart.js kütüphanesi ile SQL Server'dan gelen ham verilerin anlamlı, kavisli ve interaktif grafiklere dönüştürülmesi.

Gelişmiş Rezervasyon Sistemi: Rezervasyonların statülerine göre (Onaylandı, Beklemede, İptal) takibi ve yönetimi.

Modern UI/UX: Gradyan geçişli butonlar, modern kart yapıları ve kullanıcı dostu bir Admin Paneli.

🛠️ Kullanılan Teknolojiler
Backend: ASP.NET Core 8.0/9.0 Web API

Veritabanı: MSSQL Server (Entity Framework Core - Code First)

Mimari: Multi-Layer Architecture (Çok Katmanlı Mimari)

Yapay Zeka: GroqCloud API / LLM Entegrasyonu

Frontend: HTML5, CSS3, JavaScript, Bootstrap

Araçlar: AutoMapper, DTO Pattern, ViewComponents, Newtonsoft.Json
